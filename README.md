# 📝 Python Todo App (DevOps Project)

## 📌 Objective
This project is a simple Python To-Do application built using Flask.  
It is containerized using Docker and managed with Docker Compose.

---

## 🚀 Features
- Add new tasks  
- View all tasks  
- Delete tasks  
- Mark tasks as completed  
- Attractive GUI (Bootstrap + Mobile UI)

---

## 🛠️ Technologies Used
- Python (Flask)  
- HTML, CSS, JavaScript  
- Docker  
- Docker Compose  
- Git & GitHub  

---

## 📂 Project Structure
```
python-todo-app/
│
├── app.py
├── requirements.txt
├── Dockerfile
├── docker-compose.yml
└── templates/
    └── index.html
```

---

## ▶️ How to Run the Application

### 1. Build and Run using Docker
```
docker compose up --build
```

### 2. Open in Browser
```
http://localhost:5000
```

---

## 📸 Output
The application provides a user-friendly interface where users can:
- Add tasks  
- Delete tasks
